## what is this 😭

okay so basically.

you know those days when your WiFi dies 5 minutes before a job interview? or you say "you too" when the waiter says enjoy your meal? or your code somehow works and you have absolutely no idea why and you're scared to touch it?

**this app turns that into a comic strip.**

type what happened → get a 3-panel animated comic → share it → feel 3% better about your life choices.

that's it. that's the whole thing. it took way longer to build than it should have. no regrets.

---

## how it works (roughly)

```
you: *types something awful that happened*

melting: ah yes. i have studied the ancient texts.
         i know this pain. let me draw it.

you: *receives animated comic strip*

you: oh my god that's exactly it
```

---

## the moods 🎭

melting is an empath. it reads your situation and picks a vibe:

- **😤 stressed** — for when work is actively trying to end you
- **🤯 unhinged** — for the socially catastrophic moments replaying at 3am
- **💥 rant mode** — for when you have a very reasonable complaint and nobody is listening
- **😂 sad but funny** — for the quiet tragedies. the alarm that didn't go off. again.

each mood gets its own animated character and its own meme template drawn fresh in the browser. the "this is fine" dog has your actual situation in the speech bubble. it's very personal. it's very funny. it hurts a little.

## built with

- html. css. javascript. suffering.
- zero frameworks (we don't need them and frankly they scared us)
- canvas api for drawing memes in real time like some kind of unhinged artist
- svg animations for the little guys who act out your pain
- localstorage for the gallery because databases are someone else's problem
- vercel for deployment because we are not animals

## run it yourself

```bash
git clone https://github.com/YOUR_USERNAME/melting.git
cd melting
# open index.html
# or just use live server in vscode like a normal person
```

no npm install. no .env file. no docker. no crying (about setup).
